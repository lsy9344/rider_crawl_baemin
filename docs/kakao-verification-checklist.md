# 카카오톡 다중 창 전송 검증 체크리스트

`크롤링1`~`크롤링9` 각 탭이 자기 탭에 설정된 카카오톡 채팅방 창으로만 전송하는지 확인하는 운영자용 절차입니다. 이 검증은 KakaoTalk PC 앱이 실행·로그인된 Windows 환경에서만 수행할 수 있습니다.

## 사전 준비

- [ ] KakaoTalk PC 앱을 실행하고 로그인한다.
- [ ] 테스트용 채팅방 2개 이상을 각각 **별도 창**으로 더블클릭해 열어 둔다.
  - 창 제목이 곧 `카카오톡 채팅방명`과 정확히 일치해야 한다.
  - 가장 안전한 운영은 각 채팅방을 미리 별도 창으로 열어 두는 것이다. 그러면 메인창 검색 호환 경로를 타지 않는다.
- [ ] 같은 이름의 채팅방 창이 중복으로 열려 있지 않은지 확인한다.
- [ ] 각 탭의 `CDP 주소`가 `http://127.0.0.1:<포트>` 형태의 로컬 주소이고 탭마다 포트가 다른지 확인한다.
  - 원격(비로컬) CDP 주소는 다른 로그인 세션을 읽을 수 있어 시작/실행 시 차단된다.
- [ ] 같은 배민 센터명을 여러 탭에서 쓰면 탭마다 서로 다른 `배민 센터 ID`를 넣어 계정을 구분한다.

## 1단계: 메시지 생성만 확인 (전송 끄기)

- [ ] `크롤링1`과 `크롤링2`에 서로 다른 `카카오톡 채팅방명`을 입력하고 `전송 방식=카카오톡`으로 둔다.
- [ ] 두 탭 모두 `메시지 전송`을 끈 상태로 각 탭을 한 번씩 실행한다.
- [ ] 생성된 미리보기 메시지에 각각 `[크롤링1]`, `[크롤링2]` 라벨이 들어 있는지 확인한다.

## 2단계: 정상 라우팅 확인 (전송 켜기)

- [ ] 테스트 채팅방에 대해서만 `메시지 전송`을 켠다.
- [ ] `크롤링1`과 `크롤링2`를 시간 간격을 좁혀 연달아 실행한다.
- [ ] 각 카카오톡 채팅방이 **자기 탭의 메시지만** 받았는지 확인한다.
  - 한 방에 다른 탭 메시지가 들어가면 라우팅 실패다. 즉시 중지하고 진단 로그를 확인한다.

## 3단계: 안전 실패 확인 (모호한 대상)

- [ ] 비슷한 이름의 채팅방 2개(예: `실적봇_A`, `실적봇_A_테스트`)를 별도 창으로 연다.
- [ ] 한 탭의 `카카오톡 채팅방명`을 어떤 창 제목과도 정확히 일치하지 않게(예: `실적봇`) 설정해 본다.
- [ ] 같은 제목 창이 2개가 되도록 만들어 본다.
- [ ] 각 경우 임의의 방으로 보내지 않고 **안전하게 실패**하는지 확인한다.
  - 제목이 정확히 일치하는 창이 2개 이상이거나 카카오톡 창 목록 조회 자체가 실패하면, 메인창 검색으로 넘어가지 않고 즉시 오류를 띄워야 한다.
  - 정확히 일치하는 창이 0개면 메인창에서 채팅방을 검색해 열어 본 뒤 다시 정확히 일치하는 창을 확인한다. 그래도 안전한 단일 창을 못 고르면 실패해야 한다.

## 진단 로그

- [ ] 각 실패 경로 후 `logs/kakao_diagnostics.log`를 확인한다.
  - 선택된 창 제목/핸들/백엔드, 후보 창 목록, 거절된 후보가 기록된다.

## 기대 동작 요약

- 입력한 `카카오톡 채팅방명`과 **제목이 정확히 일치**하는 열린 카카오톡 창 1개를 우선 선택해 전송한다.
- 정확히 일치하는 창이 아직 없으면 카카오톡 **메인 창(제목이 정확히 `카카오톡`/`KakaoTalk`)** 에서만 채팅방을 검색해 연 뒤, 다시 제목이 정확히 일치하는 단일 창인지 확인하고 전송한다(호환 경로). 메인 창이 없으면 임의의 카카오톡 창을 자동화하지 않고 실패한다.
- 제목이 정확히 일치하는 창이 2개 이상이거나 카카오톡 창 목록 조회 자체가 실패하면, 메인창 검색으로 넘어가지 않고 즉시 실패한다. 임의의 메시지 입력창으로 대체하지 않는다.
- 정확히 일치하는 창을 골랐더라도 그 창의 입력창을 찾지 못하면 다른 카카오톡 창으로 넘어가지 않고 실패한다.
- 카카오톡 전송이 실패하면 텔레그램과 동일하게 약 5초 후 빠르게 재시도한다. 공통 전송 락은 실패 시에도 풀려 다른 탭 전송을 막지 않는다.
- `크롤링N` 라벨은 메시지·로그 표시용일 뿐, 채팅방 선택 기준이 아니다.
